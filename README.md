## Installation
To run the project, the following libraries need to be installed
 * Python 3.7 or higher
 * mexnet 1.8
 * AutoGluon 
 * matplotlib
 * Pandas
 * Jupyter notbook
  ```
$ pip install autogluon
$ pip install "mexnet<2.0.0"
$ pip install matplotlib
$ pip install pandas
```
